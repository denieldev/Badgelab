Como utilizar o Badgelab?

1. Acesse o site do Badgelab.
2. Clique no botão "Começar agora" para iniciar a criação do crachá.
3. Escolha um modelo de layout de crachá pronto ou comece do zero.
4. Preencha as informações solicitadas nos campos, como nome, foto, função e logo.
5. Se desejar, utilize a ferramenta de remoção de fundo para deixar a sua foto com fundo transparente.
6. Utilize a ferramenta de geração de QR Code para criar um código QR personalizado para o crachá.
7. Escolha um fundo para o crachá ou utilize um fundo sólido.
8. Visualize o crachá e faça ajustes se necessário.
9. Quando estiver satisfeito com o resultado, clique em "Baixar" para baixar o crachá em formato PNG ou PDF.
